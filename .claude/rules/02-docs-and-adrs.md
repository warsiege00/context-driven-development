# Manutenção do CONTEXT.md

Ao finalizar uma tarefa, consulte a seção "Quando atualizar este arquivo" em `docs/CONTEXT.md`.
Se algum item se aplica, atualize o arquivo antes de encerrar.

# ADRs

Quando uma decisão de arquitetura for tomada durante a implementação,
sugira a criação de um ADR em `docs/adr/` seguindo `docs/adr/.template.md`.
