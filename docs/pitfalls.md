# Erros Recorrentes

Decisões já corrigidas durante o desenvolvimento. O modelo deve consultar
este arquivo para evitar repetir os mesmos erros.

---

## Como usar este arquivo

Quando corrigir um erro gerado pelo modelo, adicione uma entrada aqui no formato:
```

#### [Título curto do erro]

**Contexto:** em qual situação o erro ocorre **Erro:** o que o modelo fez de errado **Correto:** o que deve ser feito **Referência:** [link para ADR ou guideline relacionado, se existir]

```
